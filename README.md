# MyFlick

Sistema simples em Java para cadastrar filmes e séries com avaliação.

# Descrição

O projeto permite registrar filmes ou séries informando:

- Nome
- Tipo (Filme ou Série)
- Gênero
- Nota (0 a 10)

Os dados são armazenados em um banco de dados SQLite.

## Tecnologias utilizadas

- Java
- SQLite
- JDBC
- Git

## Funcionalidades

- Cadastro de filmes e séries
- Validação da nota (0 a 10)
- Armazenamento em banco de dados
- Listagem dos registros cadastrados
